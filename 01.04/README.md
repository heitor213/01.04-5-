# Туристическое агентство - Десктоп приложение

Приложение для управления туристическими услугами (билеты, отели, развлечения) с системой авторизации и ролевой моделью (администратор/пользователь). Позволяет добавлять, редактировать, удалять и экспортировать данные в CSV.

## Стек технологий

- Python 3.x
- Tkinter (графический интерфейс)
- SQLite3 (база данных)
- hashlib (хеширование паролей)
- regex (валидация данных)
- datetime (работа с временем)
- csv (экспорт данных)

## Как установить

1. Убедитесь, что на вашем компьютере установлен Python 3.6 или выше:
```bash
python --version
```

2. Клонируйте репозиторий или скачайте файл `travel_agency.py`:
```bash
git clone <url-репозитория>
cd travel-agency
```

3. Установите необходимые зависимости (стандартная библиотека Python не требует дополнительной установки):
```bash
pip install python-regex  # если regex не установлен
```

4. Убедитесь, что у вас есть права на запись в текущей директории (для создания БД и логов)

## Как запустить

1. Запустите приложение из терминала:
```bash
python travel_agency.py
```

2. Авторизуйтесь в системе:
   - если уже есть аккаунт
   - если нету то зарегистрируйте его
   
   - **Для регистрации нового пользователя:**
     - Нажмите кнопку "Регистрация"
     - Заполните все поля (логин, пароль, email, телефон)
     - Пароль должен содержать минимум 8 символов, 1 заглавную букву, 1 цифру и 1 спецсимвол (!@#$%^&*)
   
3. После входа откроется главное меню со статистикой и доступными действиями

## Основные возможности

### Для всех пользователей:
- Просмотр списка билетов, отелей и развлечений
- Добавление новых записей
- Редактирование своих записей
- Удаление своих записей
- Экспорт данных в CSV

### Для администратора:
- Полный доступ ко всем записям всех пользователей
- Просмотр и управление всеми данными

## Структура БД

При первом запуске автоматически создаются таблицы:
- `users` - пользователи системы
- `roles` - роли (admin/user)
- `tickets` - билеты
- `hotels` - отели
- `entertainments` - развлечения
- `logs` - логи действий

## Важно

- При удалении записи вы увидите предупреждение о "грустных котиках" - это нормально 
- Все действия пользователей логируются в файл `logs/app.log`
- После 3 неудачных попыток входа аккаунт блокируется на 30 секунд
- Email не может содержать слово "admin" в доменной части
